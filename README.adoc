= Zed AsciiDoc

This extension adds support for the link:https://asciidoc.org/[AsciiDoc] language for Zed.

It is based on link:https://github.com/louiss0/tree-sitter-asciidoc/[tree-sitter-asciidoc].

== Features

- Full syntax highlighting for AsciiDoc documents
- Automatic indentation support for blocks and lists
- Text object support for vim-style operations (e.g., `dib` for delete inner block)
- Code injection support for embedded code blocks
- Proper line comment handling

=== Contributing

When contributing changes:

1. Make changes to the appropriate `.scm` files in `languages/asciidoc/`:
   - `highlights.scm` for syntax highlighting rules
   - `injections.scm` for code injection and embedded language support
   - `indents.scm` for automatic indentation behavior
   - `textobjects.scm` for vim-style text object support
2. Verify changes manually in Zed
3. Submit a pull request
